# calculator_app

A new Flutter project.

## Getting Started


A feature-rich calculator app built using Flutter with a clean dark theme and modern UI. It includes a splash screen, advanced calculations, and a history feature.


Features

- Splash Screen with app logo and developer name
- Basic & Advanced Operations: +, -, *, /, %, decimal, double zero
- Real-time Calculation
- History Feature: View previous calculations in a popup dialog
- Dark Theme UI for smooth user experience
- Clean Code Structure with organized screens

How to Run

1. Clone the repository

git clone https://github.com/pawanbisht27/calculator_app.git

2. Navigate into project folder

cd advanced_calculator_flutter

3. Install dependencies

flutter pub get

4. Run the app

flutter run
